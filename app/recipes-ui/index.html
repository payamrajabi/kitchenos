Sign in

Auth needs a one-time config file. In Terminal, from folder app/recipes-ui, run supabase login (browser), then python3 write_supabase_local.py — it writes supabase-config.local.js using your linked project. Reload this page. Or paste Project URL + anon key from API settings into that file (see supabase-config.local.example.js). Never use the service_role key in the browser.
